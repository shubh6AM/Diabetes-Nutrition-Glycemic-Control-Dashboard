# Diabetes-Nutrition-Glycemic-Control-Dashboard
**Empowering Healthier Choices with Data-Driven Meal Planning**

*Interactive Power BI Dashboard | Real-Time Veg/Non-Veg Switch | GI < 55 Target*

---

## Overview  

A **Power BI-based analytics dashboard** designed to help **diabetic and non-diabetic individuals** make informed dietary choices by analyzing:  
- Daily caloric intake  
- Macronutrient distribution (Carbs, Protein, Fat, Fiber)  
- Glycemic Index (GI) trends across meals  
- Personalized meal plans for **Vegetarian** and **Non-Vegetarian** preferences  

Built using a **7-day structured meal dataset** (`Diabetic.csv`) covering four user groups:  
- Diabetic Active  
- Diabetic Not Active  
- Non-Diabetic Active  
- Non-Diabetic Not Active  

---

## Key Features  

| Feature | Description |
|--------|-----------|
| **Interactive Slicer** | Toggle between **Veg** and **Non-Veg** plans — visuals update instantly |
| **Meal-wise Calorie Cards** | Clear breakdown: Breakfast, Lunch, Snack, Dinner |
| **GI Trend Line Chart** | Visualize glycemic impact from morning to night |
| **Nutrient Donut Chart** | Balanced daily intake at a glance (e.g., 1445 kcal, 103g protein in Non-Veg) |
| **Group-wise GI Matrix** | Compare average GI across user segments with conditional formatting |
